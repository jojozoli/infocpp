Példatár

Minden tananyag fejetehez található néhány gyakorló feladat. A végén találhatók mit ír ki típusú feladatok. A megoldások a feldatok alatt legördítve tekintehetők meg.

1/1. Írj egy olyan min függvényt mely megadja a két paramétere közül a kisebbet. A függvény működjön int, double és const char* típusokkal.

  • Megoldás
    
				int min(int a, int b) { return (a < b)?a:b; }
				double min(double a, double b) { return (a < b)?a:b; }
				const char* min(const char* a, const char* b) { return (strcmp(a ,b) <= 0)?a:b; }

1/2. Írj egy olyan swap függvényt mely két int referenciát tud megcserélni. (Írd meg a pointeres változatot is).

  • Megoldás
    
				void swapR(int& a, int& b){
					int tmp = a;
					a = b;
					b = tmp;
				}

				void swapP(int* a, int* b){
					int tmp = *a;
					*a = *b;
					*b = tmp;
				}

1/3. Hozz létre egy Math nevű névteret, melyben létrehozod a PI konstanst és egy radiánból fokra váltó rad2deg és a fordított deg2rad függvényeket. Írj hozzá példakódot is.

  • Megoldás
    
				namespace Math {
					const double PI = 3.14159;

					double rad2deg(double rad) { return  180 * rad / PI; }
					double deg2rad(double deg) { return  PI * deg / 180; }
				}

				std::cout << "72° in rads: " << Math::deg2rad(72) << '\n';
				std::cout << "PI  in degs: " << Math::rad2deg(Math::PI) << '\n';
