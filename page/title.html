Üdv az InfoC++2 oldalán!

Ez az oldal a BMEVIIIAA00 tárgyhoz készült a C++ programozás tanulásának elősegítéséhez.

Az oldalon olyan tananyagok találhatóak, melyek a C nyelv és a programozás alapvető ismeretire építenek.


			std::string InfoCpp2::getMenu(std::string menu)
				if(menu == "Info") 		return "Követelmények, aktualitások";
				if(menu == "Tananyag") 	return "Témakörökre bontott kiegészítő tananyag";
				if(menu == "Extrák") 	return "Függelék, érdekességek, HFTest használata";
				if(menu == "Kapcsolat") return "Az oldal készítője, copyright, elérhetőség";
				return "Valami más";
			}

Az oldalon található kódrészletek szabadon felhasználhatóak, továbbfejleszthetőek.
